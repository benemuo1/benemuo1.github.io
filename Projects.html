Full-Stack Library Management System

This is a full-stack application that can be used for any library. The application is used to manage the books in a library system and for users to check out and return books.

  • Database: The database is a MySQL database that is hosted on AWS. There are tables for book information, checkout records, and user profiles. The APIs in the Flask backend allow users to access the database.
  • Backend: The backend consists of Flask REST APIs that have CRUD functions. The APIs allow book and user account data to be modified and stored in the database. I used Argon2 password hashing to ensure that access to the database is secure.
  • Frontend: The frontend uses Node.js, EJS, and HTML to call the APIs and dynamically render the data for users.

Social Media Website / Forum

I created a social media website for players of the game Valorant. The website allowed users to create a custom profile, upload posts, follow friends, and more. The website was made with wordpress and custom CSS.

Features:

  • Custom profiles
  • Follow or block users
  • Groups
  • Upload posts(videos,gifs,files,text,audio)
  • Discussion forums for each in game character
  • Blog articles for news and updates
  • Direct messages(individual and group)
